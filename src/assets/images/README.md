# Images Folder

## Instructions

Place your document page images (PNG, JPG, or JPEG) in this folder.

### How it works:

1. **Add images**: Copy 3-5 page images into this folder
2. **Naming**: Images will be loaded in alphabetical order, so name them accordingly:
   - `page-1.png`
   - `page-2.png`
   - `page-3.png`
   - etc.

3. **Supported formats**: PNG, JPG, JPEG

4. **Image requirements**:
   - Any dimensions are supported
   - All images will be locked as background
   - Canvas width will automatically adjust to the widest image
   - You can only draw on the images, not on any gray areas (horizontal scroll is locked)

### Example structure:
```
src/assets/images/
  ├── page-1.png
  ├── page-2.png
  ├── page-3.png
  ├── page-4.png
  └── page-5.png
```

If no images are found, the app will display placeholder pages instead.
